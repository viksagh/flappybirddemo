Flappy Bird Demo

How to run

1. Open `index.html` in a browser. For best results run a local static server:

   In a terminal (cmd.exe) from the project folder run:

```cmd
npx http-server -p 8080
```

Then open http://localhost:8080 in your browser.

Controls
- Space / Click / Tap: flap
- R: restart after game over

Files
- index.html — game canvas and UI
- styles.css — visual styles
- game.js — game logic
